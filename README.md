## LastEmperor
a Chess960 move generator (Derived from Sapeli 1.67)

### Background
Just a move generator taken from Sapeli

### Install
1. ```make install```
2. ```lastemperor -id # Verify that LastEmperor works```
3. ```lastemperor -suite 6 # perft all 960 positions!```
